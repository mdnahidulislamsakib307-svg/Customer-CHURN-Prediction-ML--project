# Customer-CHURN-Prediction-ML--project
# 📞 Telecom Customer Churn Prediction System

## 🚀 Project Overview

This project predicts whether a telecommunications customer is likely to **churn (leave the service)** or **stay**, using a Machine Learning model.

It demonstrates a complete **end-to-end ML workflow**, including data analysis, feature engineering, model building, and deployment. Users can input customer data through a web interface and receive **instant predictions**.

## 🔢 Input Features

The model uses the following features:

- CustServCalls → Number of customer service calls  
- MonthlyCharge → Average monthly bill  
- DayMins → Daytime usage minutes  
- ContractRenewal → Contract renewal status (0/1)  
- DataPlan → Data plan subscription (0/1)
